Goals

  • Understand the medical and biological applications of computational science
  • Understand basic concepts about life: DNA, Genetic Code, proteins, biomolecules
  • Understand anatomic modeling and physiologic simulation
  • Elaborate the fundamental types of biomedical data and their organization
  • Develop in a cutting edge graphics environment using JavaScript and webGL

Program

Introduction to web programming with JavaScript

  • Introduction to collaborative development and social coding with git and GitHub
  • Basic JavaScript: Syntax, Operators, Statements, Functions, Objects
  • Advanced JavaScript: Closures, Prototype, Inheritance
  • JavaScript ecosystem: coding style guide, CoffeeScript

Biomolecules and life

  • Outline of genomics: DNA and genetic code, RNA, mapping of genes and proteins
  • Outline of proteomics: primary, secondary, tertiary and quaternary structures
  • Biochemical pathways: nervous system, neurons, neuro-muscular interface

Biomedical records, data and images

  • Electronic Medical Record (EMR) environments
  • Laboratory data, anatomic data, biomedical imaging, PACS systems, DICOM images
  • PDB (Protein Data Bank), European Bioinformatics Institute (EBI) services

Symbolic biomedical knowledge

  • Web 3.0. Ontologies for organizing biomedical data
  • The University of Washington FMA (Foundational Model of Anatomy) anatomic model

Biomedical modeling and simulation

  • Physiological models
  • Virtual Physiological Human: geometric models of proteins, cells, tissues and systems
  • Geometric models of neurons, axons and dendrites, neuro-muscular interface

Student projects

  • Design and implementation of interactive web tools for biomedical infographics
