Computational Graphics

Teachers

  • Professor Alberto Paoluzzi
  • Researcher Enrico Marino
  • Researcher Federico Spini

Goals

  • produce geometric models of highly complex components and structures
  • understand the design and development of computer-aided modeling and simulation
  • understand and apply basic and advanced computer graphics techniques
  • understand computer rendering, geometric computing and scientific visualization
  • develop in a cutting edge graphics environment using JavaScript and webGL

  • in Room N3

Program

Introduction to Web programming with JavaScript

  • Introduction to collaborative development: social coding with git and GitHub
  • Basic JavaScript: Syntax, Operators, Statements, Functions, Objects
  • Advanced JavaScript: Closures, Prototype, Inheritance

Polyhedral geometry

  • Linear and affine spaces: convex sets, affine coordinates, cones, polyhedra
  • Cellular complexes: polytopal, simplicial and cuboidal complexes
  • Convex hull, Dealunay triangulations, Voronoi complexes

Geometric programming

  • Introduction to PLaSM (Programming Language for Solid Modeling) and Plasm.js
  • First geometric constructions, Affine transformations
  • Basic computer graphics: graphical primitives, hierarchical structures, scene graphs

Introduction to Geometric Computing

  • Parametric representations of Curves, Surfaces, Solids
  • Rational and polynomial maps, tensor product patches, transfinite methods

Graphics rendering

  • 2D and 3D pipelines, projections, 3D reconstruction
  • Advanced techinique: shading, texture mapping, materials and illumination models

WebGL programming

  • WebGL frameworks, buffers, array data, uniform and attribute variables
  • Shaders, textures, lighting, blending, advanced effects

Student projects

  • Design and implement a project in the area of biomedical infographics

Teaching Materials

  • Lecture notes, Example, Exercises
  • A. Paoluzzi, Geometric Programming for Computer-Aided Design, Wiley, 2003
  • E. Marino, F. Spini, JavaScript crumbs for beginners, 2012
